# React Admin Dashboard

A modern Admin Dashboard built with **React**, **TypeScript**, and **Vite**.
The project demonstrates CRUD operations, form validation, internationalization (i18n), and a clean, responsive user interface.

## 🚀 Live Demo

https://react-admin-dashboard-phi-ebon.vercel.app/

## ✨ Features

* User Authentication (Login & Register)
* User Management (CRUD)

  * View Users
  * Add User
  * Edit User
  * Delete User
* Form Validation using React Hook Form & Zod
* Internationalization (English / Arabic)
* RTL & LTR Language Support
* Responsive UI
* Material UI Delete Confirmation Dialog
* REST API Integration using Axios
* Protected Routes

## 🛠️ Technologies Used

* React
* TypeScript
* Vite
* React Router DOM
* Axios
* React Hook Form
* Zod
* React i18next
* Tailwind CSS
* Material UI
* MockAPI
